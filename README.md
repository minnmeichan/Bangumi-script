# Bangumi-script

个人编写的有关Bangumi的脚本


## 脚本列表

### 1. 收藏人物样式改5×2
功能：
- 修改 Bangumi 用户页面收藏人物显示布局
- 调整为 5×2 布局
- 将原来的图片与名字进行居中处理，和“我的朋友”框的样式一样
- 只对收藏人物框在右边栏的有效，如果在左边栏不生效

支持网站：
- bgm.tv
- bangumi.tv
- chii.in
- bangumi.lol

#### 安装

[安装 收藏人物样式改5×2](https://github.com/minnmeichan/Bangumi-script/raw/refs/heads/main/favorite-person-style.user.js)
