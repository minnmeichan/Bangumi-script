# Bangumi-script

个人编写的有关Bangumi的脚本（AI真是太好用了你们知道吗）


## 脚本列表

### 1. 收藏人物样式改5×2
功能：
- 修改 Bangumi 用户页面收藏人物显示布局
- 调整为 5×2 布局
- 将原来图片下的名字进行居中处理
- 只对收藏人物框在右边栏的有效，如果不在右边栏会显示错误

支持网站：
- bgm.tv
- bangumi.tv
- chii.in
- bgmmi.anibt.net
- bangumi.lol
- bangumi.rdd.moe
